# CI-CD-PIPELINE-SETUP

## Description
This project demonstrates a CI/CD pipeline using GitHub Actions to automate deployment of a simple web application.

## Features
- Automated build on push to main branch
- GitHub Actions workflow
- Simple static web app

## How it works
1. Push code to GitHub
2. GitHub Actions triggers pipeline
3. CI steps run automatically
4. Deployment simulation completes

## Tools Used
- GitHub Actions
- HTML, CSS, JavaScript

## Output
A simple web page that confirms CI/CD pipeline execution.
